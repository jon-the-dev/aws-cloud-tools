# AWS Cloud Tools

Repo template with baked in goodies such as GitHub actions, CICD components and dependabot to auto-update resources for us.

## Features

* Dependabot
  * Pip
  * Terraform
  * Docker
  * GitHub-Actions
* Github Actions
  * Pylint/Python
  * Terraform tf_fmt, tf_docs, tf_validate, tf_lint
  * Docker build & test & push
* docs

# CFN Stack Backup Tool

```shell
usage: backup_stacks.py [-h] [--regions REGIONS] [--destination DESTINATION]

Backup AWS CloudFormation stacks and templates.

options:
  -h, --help            show this help message and exit
  --regions REGIONS     Comma separated list of regions to process. If not provided, all regions are processed.
  --destination DESTINATION
                        Destination directory where backups will be stored. Defaults to the current directory.
```
